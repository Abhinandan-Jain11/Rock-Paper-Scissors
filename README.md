# ✊ Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer with real-time score updates and dynamic UI feedback.

## 🚀 Features

* 🎮 Play Rock, Paper, Scissors against computer
* 🤖 Random computer choice generation
* 📊 Live score tracking (User vs Computer)
* 🎨 Dynamic UI updates based on result
* ⚡ Instant feedback (Win / Lose / Draw messages)

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

project-folder/
│── index.html

## ⚙️ How It Works

* User selects a choice (Rock, Paper, or Scissors)
* Computer generates a random choice
* Game logic compares both choices
* Result is displayed instantly
* Scoreboard updates dynamically

## 🧠 Game Logic

* Rock beats Scissors
* Paper beats Rock
* Scissors beats Paper
* Same choices → Draw
